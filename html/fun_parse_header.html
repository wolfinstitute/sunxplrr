fun_parse_header {sunxplrr}R Documentation

Parses FITS header

Description

Parses FITS header. COMMENT, HISTORY, empty and some preserved keywords are omitted.

Usage

fun_parse_header(headerName)

Value

Character vector with parsed FITS header entries.
