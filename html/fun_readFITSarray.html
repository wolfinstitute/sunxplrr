fun_readFITSarray {sunxplrr}R Documentation

Reads FITS binary array from disc

Description

Reads 16 bit or 8 bit FITS binary array from disc.

Usage

fun_readFITSarray(zz, hdr)

Arguments

zz

binary port to file.

hdr

parsed header keyword and values vector.

Value

list with image matrix and parsed header vector.
