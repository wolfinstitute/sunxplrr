fun_read_header {sunxplrr}R Documentation

Reads FITS ASCII header from disc

Description

Reads FITS ASCII header from disc.

Usage

fun_read_header(zz, maxLines = 5000)

Arguments

zz

binary port to file.

maxLines

maximal number of header lines to search for END statement. May be increased for very large headers.

Value

Character vector with FITS header card images.
