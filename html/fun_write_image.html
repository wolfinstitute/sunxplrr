fun_write_image {sunxplrr}R Documentation

Writes FITS image to disc

Description

Writes 16 bit or 8 bit FITS image to disc.

Usage

fun_write_image(x, file, hdrlst, header)

Arguments

x

image data matrix.

file

output path and file name.

hdrlst

tibble with header keywords and values.

header

string vector with card images of the existing header and of new or modified card images.

Value

nothing.
